# CS2_Parachute
CS2 Parachute - плагин для CS2, позволяющий игрокам использовать парашют при падении, удерживая клавишу USE (E) для замедления скорости падения.

## Описание плагина: CS2 Parachute - плагин для CS2, позволяющий игрокам использовать парашют при падении, удерживая клавишу USE (E) для замедления скорости падения.

https://www.youtube.com/watch?v=zQ39v9svDm0

### Требования
1. CounterStrikeSharp API версии 1.0.362 или выше
2. .NET 8.0 Runtime

### Конфигурационные параметры:

css_parachute_enabled (true) - Включение плагина

css_parachute_decrease_vector (50.0) - Снижение скорости падения

css_parachute_linear_decrease (true) - Линейное снижение скорости

css_parachute_fall_speed (100.0) - Максимальная скорость падения с парашютом

css_parachute_teleport_ticks (300) - Тайминг обновления позиции игрока


### Консольные команды:

css_parachute_help - Показать справку

css_parachute_settings - Показать настройки

css_parachute_test - Протестировать парашют

css_plugins reload CS2Parachute - Перезагрузить конфигурацию
